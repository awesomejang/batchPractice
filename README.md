# Spring batch-Practice 
spring Batch에 대한 학습 Repository

# 개발 환경
* `Java 17`
* `spring batch 4.3.8`
* __ORM__ : `JPA`
* __Framework__ : spring boot 2.7.12
* __DataBase__ : H2

# 주요 실습 및 테스트 
* spring batch의 메타데이터 테이블
* Job, Step, Tasklet 작성법
* reader, processor, writer 흐름 분석
* 트랜잭션 관리
* chunk 사이즈에 대한 처리 과정
* Job 실행 방법에 대한 분석
* 여러 Takslet 실행 시 결과에 따른 분기처리 방법
* JPA 환경에서의 Batch 사용법
